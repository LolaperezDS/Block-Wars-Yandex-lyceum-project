# Installation libraries
#### For Windows
```
pip install -r /path/to/requirements.txt
```
#### For Linux
```
pip3 install -r /path/to/requirements.txt
```
=========
# Launch
#### For Windows
Just launch `launch.bat` )))
#### For Linux
```
python3 /PATH_TO/main.py
```
